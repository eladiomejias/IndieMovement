## Indie Movement.

Indie Movement is a web page or template for bands and blogs, is also a web page into a app using manifest.json. If you want to use in your project clone and edit some css prefixes.

* Tecnologies
  * HTML
  * CSS3
  * JavaScript


* Layouts
  * Home / Page / blogging
  * Solo layout
  * Contact form
  * Game layout


* Color Palette
  * #e3e1d1 - Brownie
  * #ff4081 - Pinky
  * #222222 - Blackie
